## Features

- **Global Hotkey**: Launch ScreenSaver mode via `Cmd + Ctrl + S` from any application (Carbon API).
- **Visuals**: Support for system-installed fonts, liquid glass effects, and customizable shadows.
- **Backgrounds**: Options for solid colors, gradients, local images (WebP/HEIC), and live website URLs.
- **Presets**: Save and manage multiple design configurations as independent presets.
- **Multi-monitor**: Display clock on the primary screen or all connected displays.
- **Localization**: Native support for English and Korean.
- **System Integration**: Launch at login, menu bar icon control, and dock icon visibility options.

## Tech Stack

- Swift / SwiftUI
- Combine (State management)
- Carbon Framework (Global hotkeys)
- WebKit (Web backgrounds)
- ServiceManagement (Login items)
