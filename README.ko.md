# macOS 플립 시계

SwiftUI 기반의 macOS용 커스텀 플립 시계 애플리케이션입니다.

[English Version](./README.md)

## 주요 기능

- **전역 단축키**: 앱이 비활성 상태일 때도 `Cmd + Ctrl + S`로 화면보호기 모드를 즉시 실행합니다 (Carbon API).
- **시각 커스터마이징**: 시스템 설치 폰트 연동, 리퀴드 글래스 효과, 입체 그림자 설정을 지원합니다.
- **배경 모드**: 단색, 그라데이션, 로컬 이미지(WebP/HEIC), 온라인 이미지 및 웹사이트 URL 배경을 지원합니다.
- **프리셋 관리**: 사용자 정의 테마를 독립적인 프리셋으로 저장하고 불러올 수 있습니다.
- **멀티 모니터**: 주 모니터 또는 연결된 모든 모니터에 시계를 동시 출력할 수 있습니다.
- **다국어 지원**: 영어 및 한국어 UI를 공식 지원합니다.
- **시스템 연동**: 로그인 시 자동 실행, 메뉴바 아이콘 제어, Dock 아이콘 숨기기 옵션을 제공합니다.

## 기술 스택

- Swift / SwiftUI
- Combine (상태 및 설정 관리)
- Carbon Framework (시스템 전역 핫키)
- WebKit (웹 기반 배경 렌더링)
- ServiceManagement (로그인 항목 등록)
